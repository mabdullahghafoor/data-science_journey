# 🎓 PyVault — Student Management System

A complete command-line Student Management System
built with pure Python, demonstrating mastery of
all Phase 1 concepts.

## Features
- Complete student CRUD (Add/View/Update/Delete)
- Marks management with grade calculation
- Real-time analytics and statistics
- DSA-powered search and sorting
- File persistence (JSON)
- Full exception handling
- Modular, OOP architecture

## Concepts Used
- Variables, Operators, Conditionals, Loops
- Lists, Tuples, Sets, Dictionaries
- Functions, Lambda, File Handling
- Exception Handling, Modules, OOP, DSA

## How to Run
```bash
python main.py
```
