# 📒 Smart Contact Vault

A command-line contact management system built with
pure Python data structures.

## Features
- Add new contacts (name, phone, email, city)
- Search contacts by name
- Update contact details
- Delete contacts
- View all contacts in formatted table
- Find contacts by city (Sets)
- View unique cities (Sets)
- Export contact summary stats

## Concepts Used
- Dictionaries (main data store)
- Lists (storing multiple values)
- Tuples (immutable contact records)
- Sets (unique cities)

## How to Run
    python contact_vault.py
