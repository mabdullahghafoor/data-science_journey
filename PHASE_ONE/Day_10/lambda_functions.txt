Lambda Function:
                A Lambda is a small, anonymous, one-line function.
Anonymous means it has no name by default.

Normal Function vs Lambda — Side by Side:

Normal Function              Lambda
───────────────              ──────
def add(a, b):               lambda a, b: a + b
    return a + b

The Formula:
lambda  parameters  :  expression
   │         │              │
   │         │              └── what to return (automatic)
   │         └───────────────── inputs
   └─────────────────────────── keyword

NOTE: ⚡ Key rule: Lambda can only have ONE expression.
No multiple lines. No if/else blocks (only ternary).
If logic is complex → use a normal function.

When to Use Lambda:

✅ Short, simple one-line operations
✅ Inside sorted(), map(), filter() — very common in Data Science!
✅ When you need a quick function without formally defining one
❌ Complex logic with multiple steps → use def instead
