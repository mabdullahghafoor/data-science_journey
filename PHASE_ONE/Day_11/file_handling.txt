In this we will drop some info regarding file handling

Every program you've written so far has one big problem:

When the program stops — ALL data disappears. Forever.

students = {"Ali": 88, "Sara": 76}
# Program ends → students dictionary is GONE
# Next time you run → start from scratch 😱

Without File Handling          With File Handling
─────────────────────          ──────────────────
Data lives in RAM only    →    Data saved to disk permanently
Lost when program ends    →    Available next time you run
Can't share data          →    Share files with anyone

File Handling = Reading from and writing to actual files
on your computer — .txt, .csv, .json and more.
