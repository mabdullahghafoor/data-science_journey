# 🎓 Student Result Engine

A command-line Python application that processes student
academic results and generates detailed report cards.

## Features
- Input marks for 5 subjects
- Calculates total, percentage, and grade
- Determines pass/fail status per subject and overall
- Generates a formatted terminal report card
- Handles invalid inputs gracefully

## Tech Stack
- Python 3.11+
- Concepts used: Variables, Data Types,
  Operators, Conditional Statements

## How to Run
    python result_engine.py
