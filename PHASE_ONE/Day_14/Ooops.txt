Everything you've written so far is called
Procedural Programming — a list of instructions
that run top to bottom.
OOP is a completely different way of thinking:


Procedural Thinking          OOP Thinking
───────────────────          ────────────
"What steps do I follow?"    "What THINGS exist in my system?"
Functions + Variables        Objects with data + behavior
Hard to scale up             Built to scale
